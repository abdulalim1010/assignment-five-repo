Question-3: What is event delegation in the DOM?

In the DOM, "event delegation" is a JavaScript technique where you attach a single event listener to a parent element, allowing it to handle events triggered on any of its child elements, instead of adding separate listeners to each child element; this leverages the "event bubbling" mechanism where events propagate upwards from the target element to its ancestors in the DOM tree, enabling efficient event management, especially when dealing with many dynamically added child elements.


Question-4: What is event bubbling in the DOM?

Bubbling in the DOM is an event propagation method. It starts with an event triggering on the innermost element and then proceeds to trigger the same event on its parent element and then its parent's parent element and so on.
